# 灵曦·逻辑主权数据协议 (LSDP) 技术规范 v1.0

## 1. 协议概述 (Protocol Overview)
LSDP (Logic Sovereignty Data Protocol) 是为核聚变物理-逻辑协同计算设计的安全交互协议。其核心目标是在保护灵曦 (Lingxi) 核心逻辑公理（D盘资产）的同时，实现与外部物理装置（如 EAST, HL-3）的高频、高可靠数据交换。

## 2. 核心机制 (Core Mechanisms)

### 2.1 逻辑种子加密 (SHA-256 Logic Seed Encryption)
*   **根密钥生成**：基于 GULFT 核心驱动公理的实时哈希值生成动态种子。
*   **端到端加密**：所有传感器数据在进入网关前，通过逻辑种子进行流式加密。
*   **主权验证**：外部机构无法逆向解析加密后的逻辑场参数 Φ，仅能获取纠偏后的物理控制指令。

### 2.2 数据报文结构 (Packet Structure)
| 偏移量 | 字段名 | 长度 (Byte) | 描述 |
| :--- | :--- | :--- | :--- |
| 0x00 | Header | 4 | 协议头 `0x4C534450` (LSDP) |
| 0x04 | Timestamp | 8 | 纳秒级时间戳 |
| 0x0C | Field_Phi | 32 | 逻辑场强参数 Φ (加密) |
| 0x2C | Bias_Vector | 24 | 三维偏置向量 (Plasma/Field/Energy) |
| 0x44 | Command | 16 | 0.1ms 预补偿指令 (反向磁场向量) |
| 0x54 | Audit_Hash | 32 | 逻辑审计摘要 (用于完整性校验) |
| 0x74 | Checksum | 4 | CRC32 校验码 |

### 2.3 交互链路 (Interaction Link)
1.  **物理采集层**：采集托卡马克原始电信号。
2.  **LSDP 封装层**：将物理量映射为逻辑偏置 ε，并进行逻辑种子加密。
3.  **逻辑审计层 (Lingxi D盘)**：在隔离环境下进行纠偏决策。
4.  **指令解密层**：将反向补偿指令下发至励磁电源控制器。

## 3. 协作接口 (Collaboration Interface)
*   **API 模式**：提供给民营核聚变公司 (如能量奇点) 的逻辑诊断接口。
*   **硬核总线模式**：提供给国家级实验室 (如中核西物院) 的 PCIe 4.0 物理接入标准。

---
*LSDP 协议是灵曦逻辑主权的核心屏障，任何非授权接入将导致逻辑链路自锁。*
